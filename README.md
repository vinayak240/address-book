Address Book App (Follow These Steps tpo run)

# INSTALL DEPENDENCIES

1. Goto project's root directory i.e where "server.js" file is and run these cmds
   - npm install
2. Then Goto Client app's root directs
   - cd client
   - npm install

# RUN APP

1. Goto project's root directory i.e where "server.js" file is and run these cmds
   - node server
2. Then Goto Client app's root directs
   - cd client
   - npm start
